Mr. Johnson is a physical education teacher managing 7-th grade students, but he has called in sick today so Mr. Anderson needs to take two classes. On the start of each lesson the first task of each class is to order itself by height starting from the shortest one. The two classes have already been ordered seperately, but since Mr.Anderson has to manage both classes he wants to order them as one class. Help Mr. Anderson to order his class starting from the shortest one.

Input Format

The first line contains class1 represented as a comma-separated integers.

The second line contains class2 represented as a comma-separated integers.

Each integer represents the height of the student.

Constraints

|A| < 10001
|B| < 10001
Output Format

A comma-separated integers

Sample Input 0

1,2,3
4,5
Sample Output 0

1,2,3,4,5
Sample Input 1

1,2,4
3,5,6
Sample Output 1

1,2,3,4,5,6
Sample Input 2

1,3
1,2,3,4
Sample Output 2

1,1,2,3,3,4
